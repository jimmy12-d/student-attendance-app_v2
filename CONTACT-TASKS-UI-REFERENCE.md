## Color Scheme

### Task Types
- **Consecutive**: Red (`bg-red-100`, `text-red-800`)
- **Warning**: Orange (`bg-orange-100`, `text-orange-800`)

### Status Colors
- **Waiting**: Amber (`bg-amber-100`, `text-amber-800`)
- **Contacted**: Blue (`bg-blue-100`, `text-blue-800`)
- **Done**: Green (`bg-green-100`, `text-green-800`)

### Actions
- **Edit**: Blue icon
- **Delete**: Red icon
- **Save**: Green icon
- **Cancel**: Gray icon

## Responsive Behavior

### Desktop (>1024px)
- Full table with all columns visible
- Summary cards in 4-column grid
- Filters in horizontal row

### Tablet (768px - 1024px)
- Table scrolls horizontally if needed
- Summary cards in 2-column grid
- Filters stack in 2 rows

### Mobile (<768px)
- Table scrolls horizontally
- Summary cards in 1-column stack
- Filters stack vertically
- Reduced padding and font sizes

## Dark Mode

All components support dark mode with:
- Dark backgrounds (`dark:bg-gray-800`)
- Light text (`dark:text-gray-100`)
- Adjusted borders (`dark:border-gray-700`)
- Preserved color badges with adjusted darkness
