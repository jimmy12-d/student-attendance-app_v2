# Aside Menu Keyboard Navigation

## Keyboard Shortcuts

### Navigation
- **Arrow Down** or **Tab**: Move to the next menu item (including submenu items when expanded)
- **Arrow Up** or **Shift+Tab**: Move to the previous menu item (including submenu items when expanded)
- **Arrow Right**: Open/expand a submenu and automatically focus the first child item
- **Arrow Left**: Close/collapse the current submenu and return focus to parent

### Activation
- **Enter** or **Space**: Activate/click the currently focused menu item or toggle dropdown
- **Escape**: Close the search (when search is focused)

## Features

1. **Smart Navigation**: Arrow keys automatically navigate through all visible menu items, including nested submenus
2. **Nested Menu Support**: Arrow Right opens submenus and auto-focuses first child; Arrow Left closes and returns to parent
3. **Circular Navigation**: When you reach the end, navigation wraps to the beginning (and vice versa)
4. **Tab Support**: Standard Tab key navigation works throughout the entire menu hierarchy
5. **Auto-Expand on Search**: Parent menus automatically expand when their children match search results
6. **Visual Focus**: Browser focus indicator shows the currently focused menu item

## Usage Tips

- Click on any menu item or press **Tab** to start keyboard navigation
- Use **Arrow Down/Up** for smooth vertical navigation through all visible items
- Use **Arrow Right** to expand a submenu - it will automatically focus the first child item
- Use **Arrow Left** to collapse the submenu and return to the parent item
- Press **Enter** to activate the focused menu item and navigate to that page
- Navigation works seamlessly with filtered search results
